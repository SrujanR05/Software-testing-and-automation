<h1 align="center">🐍 Python Mini Projects</h1>

<p align="center">
A collection of beginner-friendly Python programs demonstrating
conditional statements, input validation, exception handling,
and user authentication.
</p>

<hr>

<h2>📖 Overview</h2>

<p>
This repository contains simple Python programs designed to help beginners
understand fundamental programming concepts through practical examples.
Each project focuses on a specific concept and provides hands-on learning.
</p>

<h2>📂 Projects Included</h2>

<h3>1️⃣ Age Eligibility Checker</h3>

<p>
This program checks whether a person is eligible based on their age.
A person is considered eligible if their age is between <b>18</b> and
<b>60</b> years (inclusive).
</p>

<h4>Concepts Used</h4>

<ul>
  <li>User Input</li>
  <li>Conditional Statements (if-else)</li>
  <li>Logical Operators</li>
</ul>

<h4>Sample Output</h4>

<pre>
Enter your age: 25
Person is Eligible
</pre>

<hr>

<h3>2️⃣ Password Length Validator</h3>

<p>
This program validates a password based on its length.
A password is considered valid if its length is between
<b>18</b> and <b>60</b> characters.
</p>

<h4>Concepts Used</h4>

<ul>
  <li>String Handling</li>
  <li>len() Function</li>
  <li>Conditional Statements</li>
</ul>

<h4>Sample Output</h4>

<pre>
Enter password: mystrongpassword123456
Valid Password
</pre>

<hr>

<h3>3️⃣ Division Program with Exception Handling</h3>

<p>
This program performs division between two numbers entered by the user.
It uses exception handling to prevent errors when the denominator is zero.
</p>

<h4>Concepts Used</h4>

<ul>
  <li>Arithmetic Operations</li>
  <li>Exception Handling (try-except)</li>
  <li>User Input</li>
</ul>

<h4>Sample Output</h4>

<pre>
Enter numerator: 10
Enter denominator: 2
Result = 5.0
</pre>

<h4>Exception Example</h4>

<pre>
Enter numerator: 10
Enter denominator: 0
Error: Denominator cannot be 0
</pre>

<h3>4️⃣ Simple Login System</h3>

<p>
This project implements a basic login authentication system using a predefined
username and password. The program verifies user credentials and grants access
only when the correct information is entered.
</p>

<h4>Concepts Used</h4>

<ul>
  <li>User Authentication</li>
  <li>String Comparison</li>
  <li>Conditional Statements</li>
</ul>

<h4>Sample Output</h4>

<pre>
Enter username: admin
Enter password: 1234
Login Successful
</pre>
